# Book Flight

The purpose of this project is to view available flights and book flights.


## Installation

Use the package manager npm to install the dependencies.

```bash
npm install
```

## Usage

- clone the repository
```bash
git clone https://github.com/blVck49/clan-africa.git
```

- Run the code
```bash
npm run dev
```
